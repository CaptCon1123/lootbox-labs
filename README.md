# LootBox Labs

Welcome to our project, LootBox Labs!

## Client

### To start the front-end (client):

In the root project directory (`lootbox-labs`), run:

```
npm run client
```

## Server

### To start the back-end (server):

In the root project directory (`lootbox-labs`), run:

```
npm run server
```
